Robotics Telemetry System

Built a real-time telemetry platform for collecting, transmitting, and visualizing robot sensor and performance data.

Problem to solve

After a year's worth of working as a member of our school's combat robotics team, I watched our robot get beat up in the first round of competition. Upon further inspection, we had found the culprit to be a drained battery, a sad way to go out. The next year as the captain of the team, I wanted to take this issue into my own hands. At the time we had no way of knowing whether the battery was drained until it was too late, so I decided to build a telemetry system that would allow us to monitor the battery voltage in real time and alert us if it was getting too low.

Overview

This project focused on building a telemetry system for our school's combat robotics team. The goal was to create a real-time data pipeline that could collect sensor readings from the robot, transmit them wirelessly to a base station, and provide a dashboard for monitoring key performance metrics during testing and competition.

Demo

Technical Implementation

  • Arduino Nano running main control logic
  • Bluetooth communication for wireless data transmission
  • Android dashboard for real-time visualization
  • 2 analog inputs from 0V to 12V using 16-bit analog-to-digital converter
  • Custom PCB for integrating all components

Challenges

One of the biggest challenges and learning points was integrating the hardware and software components into a cohesive system. I had to design a custom PCB to connect the sensors, microcontroller, and Bluetooth module, while also developing the software for data collection, transmission, and visualization. I also had to test that the BLE communication was reliable and that the dashboard provided accurate and timely information during robot operation.

Results

The final system allowed us to monitor the robot's battery voltage in real time during testing and competition, which helped us avoid unexpected shutdowns due to drained batteries. The telemetry platform also provided insights into other performance metrics, allowing us to make informed adjustments to our robot's design and strategy. When we used this device in the final district competition, we were able to better allocate our battery power and ended up winning the district championship, the first time in school history.

What I Learned

This project improved my understanding of instrumentation, real-time system monitoring, communication pipelines, and how better observability can make complex systems much easier to debug.
